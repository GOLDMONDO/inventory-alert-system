# Inventory Alert System

This project sends email alerts when inventory levels fall below the minimum threshold. It reads inventory data from a CSV file, detects low stock items, and sends a single email with all problematic products. It also logs all alerts to a CSV file.

## Features

- Reads inventory data from CSV
- Automatically detects low stock items
- Sends a single email with all low-stock products
- Logs all alerts to a CSV file
- Runs automatically on a daily or weekly schedule
